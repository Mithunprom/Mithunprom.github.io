Sentiment Analysis of Tweet Data Using BERT

In this project, sentiment analysis of the tweet data is done by using Bidirectional Encoder Representations from Transformers (BERT). These tweets have multilevel and multiclass. BERT is the state of the art method at the moment for the NLP. With the help of BERT, we added the output layer with the large vocabulary based pretrained BERT model and then fine tuned the model. The large case BERT is used in this project. The performance is better compared to other popular methods like RNN, LSTM, GRU.

Binary Classification for Imbalanced data by Machine Learning Model

A classification predictive model is created in this project with the response being whether the salary is in the top quartile or not. The dataset have a imbalanced class, , meaning one class have very few data points compared to other class. We used different techniques to handle the imbalanced class like using SMOTE or upsampling or down sampling or even taking into consideration of different evalution matrices. We fitted different types of machine learning models to get the best predictions. Also, we used different evaluation metries like F1 scores, ROC-AUC to describe the which model works best.
